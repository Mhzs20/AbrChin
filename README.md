# AbrChin Homepage

هوم‌پیج تمام‌صفحه و داشبوردی ابرچین؛ طراحی‌شده بر مبنای نیاز کاربر، نه نام تکنولوژی.

## اجرا

```bash
npm install
npm run dev
```

سپس `http://localhost:3010` را باز کنید.

## ساخت نسخه نهایی

```bash
npm run build
npm start
```

نسخه توسعه و Production هر دو به‌صورت پیش‌فرض روی پورت `3010` اجرا می‌شوند.
